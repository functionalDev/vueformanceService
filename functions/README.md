## Build Setup

``` bash
# install dependencies
$ npm install # Or yarn install

# serve with at localhost:5000/card3game/us-central1/api/

$ npm run serve

# deploy to firebase
$ npm run deploy

# get logs from firebase functions
$ npm run logs
```
